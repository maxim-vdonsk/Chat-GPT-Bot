# Chat GPT Bot

Telegram-бот на основе GPT-4, который умеет:
- отвечать на вопросы (текст + голос)
- генерировать и анализировать изображения
- искать информацию в интернете
- переводить текст в речь
- создавать вариации изображений

## Установка

### Шаг 1 — Клонируйте репозиторий

```bash
git clone https://github.com/your-username/chat-gpt-bot.git
cd chat-gpt-bot
```

### Шаг 2 — Создайте виртуальное окружение

Виртуальное окружение изолирует зависимости проекта от системного Python.

```bash
python -m venv venv
```

Активируйте его:

- **Windows:**
  ```bash
  venv\Scripts\activate
  ```
- **macOS / Linux:**
  ```bash
  source venv/bin/activate
  ```

### Шаг 3 — Установите зависимости

```bash
pip install -r requirements.txt
```

## Настройка

### Шаг 1 — Получите токен бота

1. Откройте Telegram и напишите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям — введите имя и username бота
4. BotFather пришлёт **токен** вида: `1234567890:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

### Шаг 2 — Узнайте свой Telegram ID (для прав администратора)

Напишите боту [@userinfobot](https://t.me/userinfobot) — он пришлёт ваш числовой ID.

### Шаг 3 — Создайте файл `.env`

Создайте файл `.env` в корневой папке проекта:

```
TELEGRAM_BOT_TOKEN=ваш_токен_бота
ADMIN=ваш_telegram_id
```

**Пример:**
```
TELEGRAM_BOT_TOKEN=1234567890:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
ADMIN=987654321
```

> Если администраторов несколько, перечислите их ID через запятую:
> ```
> ADMIN=111111111,222222222
> ```

> Файл `.env` содержит секретные данные — **не загружайте его в публичный репозиторий!**

## Запуск

```bash
python gpt_bot.py
```

База данных (`chat_history.db`) и папки для медиафайлов создадутся автоматически при первом запуске.

Чтобы остановить бота, нажмите `Ctrl+C`.

## Структура проекта

```
.
├── .env                # Секретные переменные (токен, ID администратора)
├── chat_history.db     # База данных SQLite (создаётся автоматически)
├── generated_media/    # Папка для аудио и изображений (создаётся автоматически)
│   ├── voices/         # Аудиофайлы (голосовые ответы)
│   ├── images/         # Загруженные пользователями фото
│   └── variations/     # Вариации изображений
├── config.py           # Константы: пути, список моделей, голоса
├── database.py         # Класс Database: подключение к SQLite, создание таблиц
├── gpt_bot.py          # Главный файл: все обработчики, логика бота
├── keyboards.py        # Функции для построения клавиатур Telegram
├── instructions.py     # Текст инструкции (отправляется кнопкой 📖)
└── requirements.txt    # Список Python-зависимостей
```

### Как устроена база данных

| Таблица        | Что хранит                                           |
|----------------|------------------------------------------------------|
| `profiles`     | Имя, дата регистрации, счётчики запросов             |
| `user_settings`| Выбранная AI-модель для каждого пользователя         |
| `history`      | История диалогов (вопрос + ответ + сессия)           |
| `models`       | Список доступных AI-моделей и их статус              |
| `user_stats`   | Ежедневная статистика по типу запроса и модели       |

## Частые ошибки

### `TELEGRAM_BOT_TOKEN не установлен`

Убедитесь, что файл `.env` существует в корне проекта и содержит строку:
```
TELEGRAM_BOT_TOKEN=ваш_токен
```

### Бот не отвечает на сообщения

- Проверьте, что бот запущен (`python gpt_bot.py`).
- Убедитесь, что в консоли нет ошибок.
- Попробуйте отправить `/start` в Telegram.

### Ошибка при генерации изображения / аудио

Библиотека `g4f` использует сторонние провайдеры, которые иногда недоступны. Попробуйте повторить запрос — встроен механизм 3 повторных попыток.

### Кнопки администратора не видны

Убедитесь, что ваш Telegram ID указан в переменной `ADMIN` в файле `.env`. Узнать свой ID можно через [@userinfobot](https://t.me/userinfobot).
